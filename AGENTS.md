# Instruções de Idioma e Localização

Para este projeto, todas as interfaces, mensagens de erro, labels, menus e qualquer conteúdo gerado pela IA (exceto quando solicitado especificamente o contrário pelo usuário para fins de tradução de conteúdo) DEVEM estar em **Português (BR)**.

## Diretrizes de Tradução
- Mantenha termos técnicos comuns se não houver tradução direta adequada, mas prefira o equivalente em português.
- "Short Video" -> "Vídeo Curto"
- "Post" -> "Postagem"
- "Story" -> "Story" (termo amplamente aceito no Brasil)
- "Draft" -> "Rascunho"
- "Published" -> "Publicado"

## Tom de Voz
- O tom deve ser profissional, tecnológico e épico (referenciando Nazarick/CREA.OS quando aplicável).
